# ABAP Development Skill

This skill provides comprehensive guidance for ABAP development with AWS, covering:

- **abapGit file format** — exact XML/ABAP serialization rules to avoid constant diffs
- **ABAP development cycle** — the end-to-end workflow from repo setup to activation and testing
- **AWS SDK for SAP ABAP** — session/client creation, data types, paginators, waiters, and exception handling
- **Good ABAP practices** — coding tips, style guidelines, and function group conventions

## Installation

Add this skill to your Kiro workspace by cloning or referencing:

```
https://github.com/frij-aws/guidance-for-deploying-sap-abap-accelerator-for-amazon-q-developer.git
```

Then add `.kiro/skills/abap-dev` to your Kiro skills path.
